---
title: 'El detective algorítmico'
---

Un programador midió su algoritmo con distintas entradas:

```
n = 100 → 0.01 seg
n = 200 → 0.04 seg
n = 400 → 0.16 seg
n = 800 → 0.64 seg
```

Cada vez que duplica n, el tiempo se multiplica por 4.
1. ¿Qué complejidad tiene este algoritmo?
2. ¿Podés deducirlo solo con los datos? Justificá tu respuesta.
